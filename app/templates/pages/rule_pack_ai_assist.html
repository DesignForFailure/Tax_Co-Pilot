How the Round Trip Works

  1. Generate the prompt here and copy it.
  2. Paste it into the AI assistant you already use (Claude, ChatGPT, a local model — your choice).
  3. Copy the AI's answer — one block with # === MANIFEST === and # === RULES === markers.
  4. Paste it into Import YAML → Paste combined YAML. The import validates everything before saving.
  5. If validation rejects it, copy the error message back to the AI and paste its corrected answer.

Privacy: this app never contacts an AI service. Only what you yourself paste into an AI chat leaves this machine — the prompt contains the rules you describe plus public rule-pack format documentation, never your return data.

Verify the result: AI-drafted tax logic must be checked against the official forms and instructions. Validation proves the pack is well-formed, not that the rates are right — review every number, then run a test calculation and read the audit trace.

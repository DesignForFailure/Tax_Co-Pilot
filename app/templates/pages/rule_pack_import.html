Importing from GitHub

To import a pack from a repository checkout or GitHub raw files:

  1. Open the rule pack directory that contains the manifest and rules YAML files.
  2. Download or copy the raw manifest.yaml file.
  3. Download or copy the raw rules.yaml file.
  4. Upload both files here and optionally assign a custom display name.
  5. Review validation errors on this page before retrying if the import is rejected.

Imported packs are always created as custom variants so they never overwrite the shipped standard logic.

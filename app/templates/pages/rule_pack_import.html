Importing from GitHub

To import a rule pack from a GitHub repository:

  1. Navigate to the rule pack directory in the repository.
  2. Download the raw manifest.yaml file (click Raw, then Save As).
  3. Download the raw rules.yaml file the same way.
  4. Upload both files above and give the pack a custom name.
  5. The pack will be validated before being saved — invalid YAML or rule errors will be shown here.

Imported packs are always created as custom variants so they never overwrite standard packs.

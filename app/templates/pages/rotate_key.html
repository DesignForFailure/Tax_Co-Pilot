Before You Continue

Use a new strong password and verify it carefully.

Rotation updates the database encryption key in place. If the new password is lost, the encrypted data becomes inaccessible.

Minimum Length
12+
Scope
Entire DB

Operational Note

Rotate the key only after the database is unlocked and stable. Interrupted rotations can leave the database unavailable until the original key or a good backup is restored.
